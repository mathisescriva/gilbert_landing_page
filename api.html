Documentation des endpoints

Découvrez les principaux endpoints disponibles via l'API Gilbert

POST /v1/transcribe

Transcrivez un fichier audio en texte structuré avec timestamps et locuteurs.

bash 
curl -X POST https://api.gilbert.fr/v1/transcribe \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "audio_url": "https://example.com/audio.mp3",
    "language": "fr",
    "speaker_diarization": true
  }'
json 
{
  "id": "trans_abc123",
  "status": "completed",
  "transcript": {
    "text": "Bonjour, bienvenue à cette réunion...",
    "segments": [
      {
        "start": 0.0,
        "end": 2.5,
        "speaker": "speaker_1",
        "text": "Bonjour, bienvenue à cette réunion"
      }
    ]
  },
  "created_at": "2025-01-15T10:30:00Z"
}
POST /v1/summarize

Générez un résumé intelligent d'une transcription ou d'un texte.

bash 
curl -X POST https://api.gilbert.fr/v1/summarize \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Texte à résumer...",
    "max_length": 500,
    "style": "executive"
  }'
json 
{
  "id": "sum_xyz789",
  "summary": "Résumé exécutif de la réunion...",
  "key_points": [
    "Point important 1",
    "Point important 2"
  ],
  "created_at": "2025-01-15T10:35:00Z"
}
POST /v1/actions

Extrayez automatiquement les actions et décisions d'une réunion.

bash 
curl -X POST https://api.gilbert.fr/v1/actions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "transcript_id": "trans_abc123",
    "extract_decisions": true,
    "extract_tasks": true
  }'
json 
{
  "id": "act_def456",
  "decisions": [
    {
      "text": "Décision prise lors de la réunion",
      "context": "Contexte de la décision"
    }
  ],
  "tasks": [
    {
      "description": "Tâche à réaliser",
      "assignee": "Personne assignée",
      "priority": "high"
    }
  ],
  "created_at": "2025-01-15T10:40:00Z"
}

Questions fréquentes

Tout ce que vous devez savoir sur l'API Gilbert

L'API Gilbert est actuellement disponible sur demande. Contactez-nous via le formulaire de contact en précisant votre projet et vos besoins. Notre équipe vous répondra dans les plus brefs délais pour discuter de votre intégration.

L'API Gilbert supporte les principaux formats audio : MP3, WAV, FLAC, M4A, OGG. Les fichiers peuvent être fournis via une URL ou uploadés directement selon votre plan d'accès.

Les limites dépendent de votre plan d'accès. Pour les plans Enterprise, nous proposons des quotas personnalisés adaptés à vos besoins. Contactez-nous pour discuter de vos volumes et obtenir un plan sur mesure.

Oui, l'API Gilbert est entièrement conforme au RGPD. Toutes les données sont hébergées en France et traitées selon les normes de sécurité les plus strictes. Nous garantissons la souveraineté de vos données.

L'API est une API REST standard qui peut être intégrée avec n'importe quel langage de programmation. Des exemples de code sont disponibles dans notre documentation. Pour les besoins spécifiques, notre équipe peut vous accompagner dans l'intégration.

Le temps de réponse dépend de la longueur du fichier audio. Pour un fichier de 10 minutes, la transcription est généralement disponible en moins de 2 minutes. Les plans Enterprise bénéficient de priorités de traitement pour des temps de réponse encore plus rapides.

Gilbert est un assistant vocal souverain qui transforme votre voix en intelligence. De la parole à l'intelligence. En toute simplicité.
